## 🇬🇧 English

This repository is the source code for my personal technical blog and research notebook. It is a high-performance system designed to bridge the gap between local scientific research workflows and web publication.

### Platform Highlights
- **IDE-Centric Interface:** The administrative dashboard utilizes a 3-panel workspace, optimized for the rapid development of technical and scientific content.
- **Hierarchical Indexing:** Unlike tag-based systems, this project utilizes a recursive file-tree navigation system. URLs map directly to the directory structure, ensuring logical content organization.
- **Obsidian Integration:** Full support for standard Markdown and native Obsidian syntax, including local image linking `![[image.png]]`.
- **Academic Rigor:** Built-in generation of APA-formatted citations and automated CC BY-NC-SA 4.0 license management.

## 🇪🇸 Español

Este repositorio contiene el código fuente de mi blog personal y cuaderno de investigación. Es un sistema de alto rendimiento diseñado para conectar flujos de trabajo de investigación científica local con la publicación web.

### Aspectos Destacados
- **Interfaz Centrada en IDE:** El panel administrativo utiliza un espacio de trabajo de 3 paneles, optimizado para el desarrollo rápido de contenido técnico y científico.
- **Indexación Jerárquica:** A diferencia de los sistemas basados en etiquetas, este proyecto utiliza un sistema de navegación mediante árboles de archivos. Las URLs mapean directamente la estructura de directorios, garantizando una organización lógica del contenido.
- **Integración con Obsidian:** Soporte completo para Markdown estándar y sintaxis nativa de Obsidian, incluyendo enlaces locales a imágenes `![[imagen.png]]`.
- **Rigor Académico:** Generación automática de citas en formato APA y gestión automatizada de licencias CC BY-NC-SA 4.0.
